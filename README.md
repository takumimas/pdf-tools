# PDF Tools

PDFファイルを操作するデスクトップアプリケーション。

## 機能

- **PDF結合**: 複数のPDFを1つに結合
- **PDF分割**: PDFを1ページずつ分割
- **PDF→JPEG**: PDFの各ページをJPEG画像に変換
- **JPEG→PDF**: 複数の画像を1つのPDFに変換
- **パスワード解除**: パスワード保護されたPDFを解除

## 技術スタック

- Tauri 2
- React + TypeScript
- pdf-lib / pdfjs-dist

## 開発

```bash
npm install
npm run tauri dev
```

## ビルド

```bash
npm run tauri build
```
